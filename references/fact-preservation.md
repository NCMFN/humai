# Fact Preservation Rules

Never modify:
* Numerical values, results, and percentages
* Author names, affiliations, datasets
* Dates and locations
* All citations and bibliography formatting
* LaTeX commands, labels, references, equations (e.g., `\cite{}`, `\ref{}`, `\label{}`, `\section`, `\subsection`)
* Numbers, units, statistical notation
* Proper nouns and named methods (e.g., XGBoost, LightGBM, ARIMAX)
* Metrics (e.g., RMSE, R², f1_weighted)
* Section numbering
